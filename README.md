# React + Vite

## Clonacion del Repositorio

```bash
git clone https://github.com/02Alexis/Instag-Clone.git

# Navega al directorio del proyecto:
cd Instag-Clone

# Instala las dependencias del proyecto
npm install

# Ejecuta
npm run dev

# Abre en tu navegador
http://localhost:5173
```
